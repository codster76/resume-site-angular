What is it?
This is a simple item tracker for Dungeons & Dragons. The silly name is an inside joke between a friend and I. It was made as a learning exercise for creating and connecting a backend to my frontend.

How it Works
It’s quite simple. I have a REST API in the backend that serves JSON containing the items to display on the frontend. I don’t have a database to use, so the items are stored in memory, meaning the data isn’t persistent. Despite this, though, all of the data serving and display functionality is there.

The API is able to handle getting, adding, deleting and modifying items.

On the frontend, retrieved items are simply displayed in various ways, with a UI for viewing, sorting, adding, editing and deleting items.
