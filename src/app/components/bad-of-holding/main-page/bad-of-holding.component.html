What is it?

This is an item tracker for Dungeons & Dragons. The silly name is an inside joke between a friend and I. This is a showcase for a UI for manipulating and managing objects in a backend system/database. This could really be tracking anything, from job tickets, to stock, to D&D items. It's got the basic functionality of adding, removing, editing, sorting and viewing objects.

How it Works

Itâ€™s quite simple. I have a REST API in the backend that serves and receives items. Changes/deletions/additions made on the frontend are reflected in the backend. For simplicity and to avoid the need for maintenance, there isn't a real database tracking the items for this site and it's just simulated in the server's memory. Adding a real database would certainly be possible, but it would not meaningfully change the functionality being demonstrated on this page.
